# 🎸 Understand Guitar

> Personal notes and reference material organised for long-term study and quick lookup.

---

## What this is

This repository is a structured knowledge base of lesson notes that teaches the **logic and theory behind music**, not just where to put your fingers.

The core philosophy of the course (and this repo):

> *"Music is a language. The guitar is a typewriter. The music comes from your mind."*

Notes are written in Markdown so they render cleanly on GitHub, are fully searchable, and can be read on any device.
